This app is a static personal portfolio site with no database requirements. It includes a hero section, about paragraph, three project cards in a responsive grid, and a contact form. The contact form is processed by a Vercel serverless function (`api/contact.ts`) owned by the Email Agent, which uses Resend for email delivery. No Supabase tables, RLS policies, storage buckets, or edge functions are used. Features like form validation, honeypot, rate limiting (via Upstash Redis), and error logging (to Sentry) are handled at the edge by the serverless function. The frontend is built with React + Vite + Tailwind, following a warm minimalist design system (Fraunces/Satoshi fonts, beige background, dark green and burnt orange accents).
